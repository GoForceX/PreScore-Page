关于《出分啦》的未来

《出分啦》已经在新的一届维护团队中已经维护了将近一年。首先要感谢项目的原作者 GoForceX 提供的大力帮助。从技术角度总结，在过去的一年，我们使用 Rust 重构了后端的实现，并完成了数据的迁移，并在后端对部分内容进行了优化。前端在保证可用性的同时，增加了 分布（查询成绩大致分布）、组合（分组）、更多的学科（神秘接口实现）。在实现本项目的同时，开发团队也从中提高了自己的开发水平。

在过去的一年，官方对我们所知的技术途径进行封堵，许多内容可能会 临时 / 永久 不可用，我们也在尽力的寻找替代解决方案，以来尽量保证可用性。不过最终许多功能也无法恢复，我们对此深感抱歉。

​~~其次，支持设备范围的问题，我们可能会考虑在近期发布的版本中增添桌面版 (windows / mac / linux) 。请关注项目官网 / Github Repo 的信息。~~

​不过由于当前开发组精力有限，当前开发组已无力保证基础功能（登录/查询成绩/查询小分/估分）的可用性，同时也没有找到可靠的下届维护成员。所以我们在此向大家告知，我们的维护工作即将停止。

​最终，我们将发布最终版（1.0）。在此之后，我们不再保证任何功能的可用性（除非找到合适的新任维护成员）。同时，估分功能将至少维护到新学年前（2025.09）。在此，也希望大家能协助我们找到懂技术开发的同学，若有兴趣，欢迎来私信我以来了解更多。再次感谢大家使用《出分啦》。
